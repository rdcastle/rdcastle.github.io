ITSM

Information Technology Service Management, 기존의 IT 관리 시스템(V2)이 기능 및 기술향상에 초점을 두었다면, ITSM(V3)은 프로세스와 고객 요구사항 및 편의성에 초점이 맞춰져 있다
즉, 정보시스템 사용자가 만족할 수 있는 서비스를 제공하고 지속적인 관리를 통해 서비스 품질을 유지, 증진시키기 위한 일련의 활동을 의미한다(서비스 관점, 고객 중심 IT 서비스 관리)

ITIL

IT Infrastructure Library, 세계적으로 공인된 Best Practice(모범 실무)를 제공하는 IT서비스 품질모델. IT 서비스 제공 및 관리에 대한 현존하는 유일한 포괄적, 공개적 가이드로 ITSM 분야의 업계 최고 실행 지침서이다. ITIL의 프레임워크(기본개념구조, 뼈대)는 Service Delivary와 Service Support로 분류되며, 이 두 핵심 프레임워크는 ITIL 기반의 ITSM 을 구성하는 기본 요소로 활용되고 있다.

ISO 20000

국제표준화기구 ISO 가 2005년 발표한 IT 서비스 관리(ITSM) 국제 표준

SLA

Service Level Agreement, 서비스 수준 협약, 서비스 사업자와 사용자 간의 협약. 제공될 정보서비스 및 그와 연관된 여러 조건들에 대한 서로의 책임과 의무사항 기술

증강현실

Augmented Reality, AR, 현실세계 + 가상세계

모바일 컴퓨팅

Mobile Computing, 무선 이동통신과 PDA, 인터넷을 통해 컴퓨터와 통신기술을 효과적으로 연계시켜(모바일 기기) 언제 어디서나 이동하면서 정보교환, 수집, 검색, 정리, 저장하는 기술. 즉, 휴대용 기기로 자유롭게 네트워크에 접속하여 업무를 처리할 수 있는 환경을 제공하는 것

SCM

Supply Chain Management, 공급사슬관리, 공급유통망관리, 유통 공급망에 참여하는 모든 업체(원자재, 제조, 도소매 등) 들이 협력을 바탕으로 정보기술을 활용, 불필요한 시간과 비용을 절감하려는 관리 기법. 정보기술 기반 모든 유통라인의 유기적 통합

플래시 메모리

Flash Memory, RAM 과 ROM 의 장점을 규합한 메모리. 데이터저장, 빠른 처리속도, 대용량 반도체에 저장돼 작고 가볍다. USB 는 Flash Memory 를 사용한 저장장치이다.

Nomophobia

스마트폰 중독, 없으면 초조하고 불안감을 느낀다

IMS

IP Multimedia Subsystem, 인터넷 프로토콜 기반 멀티미디어 서비스 제공 규격. 3GPP 에서 고안한 네트워크 구조(Framework). IMS 의 대표적인 예로는 MMTel (Multimedia telephony)이 있다

NDN

Named Data Networking, 서버와 IP 주소 기반이 아닌 콘테츠 자체를 네트워킹의 주체(Entity)로 삼는 미래의 인터넷 아키텍쳐

Social Dining

SNS 공통 관심사의 사람들끼리 식사(1인가구)

AMR

Automatic Meter Reading, 네트워크 내 각각의 단말기를 통해 검침. 가스검침의 경우 각 단말기는 한 가정을 의미하고 방문 없이 검침 및 고지서 자동발급, 수요패턴을 관리한다

Foursquare

데이스 크롤리 - 위치 기반 소셜 네트워크, 땅따먹기!

Infodemics

Information + epidemics, 정보전염병, 잘못된 정보의 빠른 확산으로 대중의 두려움이 증폭되는 현상
