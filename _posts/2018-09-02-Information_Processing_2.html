  1. Smoke Testing

    2. 기초안정성 검사, SW 일부를 개발한 후 본격적인 시험 수행에 앞서 SW의 결함을 식별하고 변경의 유효성을 검사하는 시험의 형태. 시스템 핵심 기능들에 대해 엄격하게 시험하는 것이 아닌 안정성 측면에서 주요 기능이 잘 동작하는지 확인하기 위함. 즉, 제품 품질의 본격적인 테스트 에는 많은 시간과 비용이 투자되는데, 시스템의 기초적인 결함으로 인해 구체적인 기능을 테스트하지 못하는 비효율적인 자원낭비가 초래될 수 있으므로 그것을 사전에 방지하기 위한 기초검사이다.

  2. Digital Footprint

    3. 웹페이지 접속이나 온라인 결제정보 등 사람들의 온라인 활동 기록을 지칭

  3. Ransomware

    4. 인터넷 사용자의 컴퓨터에 잠입해 내부 문서나 스프레트시트, 그림파일 등을 암호화해 열지 못하도록 만든 후 금품을 요구하는 악성 프로그램. 사용자의 문서를 '인질'로 잡는다는 의미.

  4. POSIX

    5. Portable Operating System Interface, 포직스, 이식 가능 운영체제 인터페이스의 약자로 UNIX OS 끼리의 이식성 및 호환성을 위해 공통의 API 를 정리한 것(규격, IEEE).

    + API: Application Programming Interface, OS 와 SW 간 통신을 위한 언어나 메시지 형식, 응용 프로그램을 이용해서 운영체제나 프로그래밍 언어가 제공하는 기능을 제어할 수 있게 만든 인터페이스
