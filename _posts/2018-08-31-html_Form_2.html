input 태그

사용자 입력을 받아들이는 태그. 다양한 속성이 존재하며, 그 중 type 속성은 다양한 input 형태를 구현한다.

  • type : text, radio, checkbox, password, button, hidden, fileupload, submit, reset ...

    • 위와 같은 속성값을 가지며, 각각의 속성값은 input 태그의 의미나 형태를 규정한다

그 외에도 input 태그를 표현하는 다양한 속성들이 있다

  • name : 태그 이름 지정
  • readonly : 읽기 전용으로
  • maxlength : 최대 글자 수 지정
  • required : 해당 태그가 필수태그로 지정됨. 이것을 입력하지 않고 submit 하면 에러메시지 출력됨(HTML5)
  • autofocus : 웹 페이지가 로딩되자마자 해당 태그로 포커스 이동(HTML5)
  • placeholder : 태그에 입력할 값에 대한 힌트(HTML5)
  • pattern : 정규표현식 사용, 유효한 값을 지정(HTML5)
