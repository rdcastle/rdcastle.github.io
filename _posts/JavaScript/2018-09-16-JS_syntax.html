개요

자바스크립트, JQuery 에 이르기까지 '윤인성' 저자의 [모던 웹을 위한 JavaScript jQuery 입문(ECMAScript 5/6, jQuery 3.X 대응)] 책을 조금씩 따라간다.

기본문법

자바스크립트 코드는 html 을 뼈대로 한 전체 코드 중 'script' 태그 내에 속한 구문들로 구성된다.

자료형

  • 문자열 자료형 - String

    • 문자열 자료형은 따옴표나 쌍따옴표 중 아무거나 사용해도 되지만, 일관성을 유지하는 것이 좋다. 만약 문장 내에 따옴표가 필요하다면, 특수문자도 마찬가지지만, 이스케이프 문자를 활용한다(기타 모든 특수문자 및 개행은 이스케이프를 활용하는 것 기억할 것). 아니면 쌍따옴표 내에서는 홑따옴표, 홑따옴표에서는 쌍따옴표를 사용해도 된다
    • 여러 개의 문자열 자료형을 연결하려면, 자바에서와 같이 덧셈 기호를 이용한다.

  • 숫자 자료형 - Number

    • 숫자 자료형은 정수나 유리수의 구분 없이 모두 같은 자료형으로 인식한다
    • 기본적인 사칙연산이 가능하다. 나머지 연산도 가능하며, 사칙연산 시에는 연산자 우선순위를 고려해야 한다

  • 불 자료형 - Bool

    • 불 자료형은 대표적인 논리 자료형이다. 비교연산이나 논리연산 시 사용한다
    • 출력값은 'true' or 'false' 이며, 불 자료형 간 비교연산 시에는 true 는 1 로, false 는 0 으로 변환하여 출력한다
    • '!', '&&', '||' 등 논리연산도 가능하다
    • 비교연산과 논리연산

      '30 > 20 > 10' 과 같은 단순나열식 비교연산은 원하는 값을 출력하지 않는다(연산자 우선순위에 따라 false 출력). '30 > 20 && 20 > 10' 과 같이 논리연산으로 바꾸면 원하는 true 값을 출력한다

  • undefined 자료형

    선언하지 않는 변수 또는 변수를 선언했지만 초기화하지 않았을 때 자료형은 'undefined' 이다. undefined 자료형은 굉장히 많은 곳에 활용된다고 한다..

  • 기타 자료형 - 함수형, 객체형

    추후 확인!!

  • 자료형 확인하기

    확인하고자 하는 식별자나 변수를 typeof() 연산자에 투입하면 된다. 그것을 alert 로 출력하면 해당되는 자료형이 출력된다

변수

  • JavaScript 에서 변수는 
    
              <script>
                var [변수명];
              </script>

    형태로 선언한다
  • var 는 구체적인 자료형을 구분하지 않으며 문자열, 정수, 실수 모두 수용한다. 초기화 시에 동적으로 Type 을 결정하기 때문
  • 초기화는 변수 선언 시 혹은 나중에 할 수 있으며(할당), 일반적으로 선언 시에 한다. 한 번에 여러 변수 선언 가능하다
  • 문자열의 경우에도 재할당이 편하며, 문자열을 추가해서 출력하려면 alert 함수에서 더하기 연산을 하면 된다

변수 추가

자바스크립트에서 'var' 로 선언되는 변수는 전역 개념으로 활용된다. 그것이 어느 지역에 있든 말이다. 그러나 때로 제한적인 용량을 차지하거나 필요에 따라 생성되었다 소멸하는 변수를 선언할 경우가 생기는데, 이때 활용하는 선언 데이터타입은 'let' 이다. 사용법은 var 와 같다. 다만, 지역적 개념(해당 스코프 내에서만)으로 사용 가능하다. 추가적으로 'const' 키워드는 상수를 선언할 때 사용한다. let 과 const 는 버전에 따라 적용되지 않는 브라우저도 있다(ECMAScript 6 부터 적용된 개념임)

기본출력 함수 alert();

해당 페이지에서 alert 함수가 실행되면, 매개변수로 받은 메시지를 별도의 알림창을 통해 화면에 출력한다. 활용예는 다음과 같다

        <script>
          alert([String message]);      // 기본문법
          alert('Hello JavaScript!!');  // 활용예
        </script>

기본입력 함수 - prompt();, confirm();

JavaScript 에서 사용자 입력함수 prompt() 는 문자열 자료형만을 입력받는다. 그러므로 사용자가 숫자를 입력한다면 그것을 문자열로 인식하며, 문자형을 숫자형으로 변환하는 과정이 필요하다. prompt() 함수를 구현할 때는 함수를 매개변수와 함께 호출한 뒤 변수에 저장하여 그것을 출력한다. 매개변수로는 [입력요청 메시지], [디폴트 메시지] 를 필요로 한다 

          <script>
            var input = prompt('Message', 'DefStr');  // 함수호출 및 변수에 할당

            alert(input);    // 해당 변수를 출력
          </script>

이렇게 입력된 문자열은 함수에 의해 반환된 후 변수에 그대로 저장되며, 해당 변수는 입력한 문자열을 저장한 문자열 변수가 된다. 최종적으로 값을 반환하는 함수의 이러한 특성을 '리턴' 이라고 한다

confirm() 함수는 사용자로부터 불 자료형을 입력받는다. 방식은 prompt() 함수와 유사하다. 다만 사용자가 입력요청 메시지에 대해 예/아니오 로만 응답할 수 있을 뿐이다. 사용자가 예(확인)을 누르면 true 를, 아니오(취소)를 누르면 false 를 리턴한다 

          <script>
            var input = confirm('수락하시겠습니까?');  // 함수호출 및 변수에 할당

            alert(input);    // 해당 변수를 출력
          </script>

함수의 최종반환값은 불 형이므로, 해당 변수는 불 형이 된다

숫자와 문자열 자료형 변환(불 추가)

  • 자동 형변환 - 데이터 연산 시 숫자형과 문자열을 더하기 연산하면 숫자형이 문자열로 자동 형변환되어 계산되며, 더하기를 제외한 사칙연산에서는 문자열 자료형을 숫자형으로 자동 형변환 한다. 참고할 점은 숫자가 문자열로 저장된 두 데이터(즉 문자열 끼리)를 더하기 연산하면 문자열로 더해지지만, 더하기를 제외한 사칙연산을 하면 모두 숫자형으로 변환되어 계산된다
  • 명시적 형변환 - prompt() 함수에서 사용자 입력시에는 무조건 문자열로 저장되므로, 사용자 입력 숫자를 저장하려면 문자열 자료형을 숫자형으로 형변환하는 과정이 필요하다. 이 때는 명시적 형변환이 필요한데 Number() 함수나 String() 함수를 사용한다. 말 그대로 다른 자료형을 숫자형, 문자열 자료형으로 변환하는 함수이다. 추가적으로, 만약 숫자가 아닌 값을 Number() 함수에 입력하면, NaN(Not a Number) 값을 출력한다 (자바스크립트로 나타낼 수 없는 숫자를 의미. 예를 들어 복소수는 표현하지 못한다)
  • 불 자료형 변환에는 Boolean() 함수가 사용된다. 사용법은 Number(), String() 함수와 같다. 다만 입력값이 0, NaN, 빈 문자열(''), null, undefined 이 다섯가지 경우에는 false 로 변환되고 나머지는 true 로 변환된다(따옴표로 감싸진 문자열인지 주의해서 확인할 것)
    또한, 조건문이나 논리 부정 연산자를 사용하면 Boolean() 함수 없이도 불 형으로 변환된다

일치 연산자

자바스크립트는 데이터 사용 및 형변환에 있어서 굉장히 유연하다. 자동 형변환이 이루어지다보면 의도하지 않은 결과가 나오는 경우가 발생하는데, 이것을 예방하기 위해 사용하는 것이 일치연산자이다. '===' 과 '!==' 이 일치연산자이다. 다음 예제를 보자

          <script>
            alert('' == false);
            alert('' == 0);
            alert(0 == false);
            alert('273' == 273);
          </script>

위 네 가지 모두 true 를 출력한다. 자동 형변환 때문인데, 이것을 막고 원하는 자료형을 확실하게 구분짓고 싶다면 일치 연산자를 사용한다 

          <script>
            alert('' === false);
            alert('' === 0);
            alert(0 === false);
            alert('273' === 273);
          </script>
