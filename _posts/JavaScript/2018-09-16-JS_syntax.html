개요

자바스크립트, JQuery 에 이르기까지 '윤인성' 저자의 [모던 웹을 위한 JavaScript jQuery 입문(ECMAScript 5/6, jQuery 3.X 대응)] 책을 조금씩 따라간다.

자바스크립트 코드는 html 을 뼈대로 한 전체 코드 중 'script' 태그 내에 속한 구문들로 구성된다.

기본출력 함수 alert();

해당 페이지에서 alert 함수가 실행되면, 매개변수로 받은 메시지를 별도의 알림창을 통해 화면에 출력한다. 활용예는 다음과 같다

        <script>
          // 문법
          alert([String message]);
          // 활용예
          alert('Hello JavaScript!!');
        </script>
