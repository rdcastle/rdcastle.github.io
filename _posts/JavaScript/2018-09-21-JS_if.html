조건문

조건문에 관한 모든 내용은 JAVA 와 동일하고, 다만 간편해지거나 추가되는 내용이 있다

삼항연산자

if 문 또는 switch 문 대신에 매우 간랸하게 활용할 수 있는 조건문이다. 기본적인 형태는 다음과 같다 

        [불 표현식] ? [참일 때 실행하는 문장] : [거짓일 때 실행하는 문장]
삼항연산자는 코드가 너무 복잡해질 수 있으므로 한 줄로 표시할 수 있을 때만 사용하는 것이 좋다

짧은 조건문

논리 연산의 기본적인 원리를 이용한 것으로, 'OR 연산' 의 경우 전자가 'true' 라면 후자를 체크하지 않는 특성이나 'AND 연산' 의 경우 전자가 'false' 라면 후자를 체크하지 않는 특성을 활용한다. 한 마디로 '논리 연산 과정에서 어떤 항은 아예 체크하지 않고 넘어갈 수 있다'는 특성을 이용하는 것이다. 반대로 꼼꼼하게 논리적으로 따져보지 않으면 예상치 못한 결과를 도출하기도 한다

Date 클래스

현재 년도, 날짜, 시각, 분초 등의 정보를 얻기 위한 클래스이다. 유용한 메서드로 getFullYear(), getHours(), getDate(), getDay() 등이 있다

indexOf() 메서드

        [문자열].indexOf([검사문자열]);
형식으로 사용되며, 앞에 있는 문자열에 뒤에 있는 검사문자열이 포함되어 있을 경우 그 위치를 출력한다. 만약 포함되어 있지 않으면 -1 을 리턴한다

반복문

조건문과 마찬가지로 JAVA 와 동일하면서 간편화되거나 추가되는 내용이 있다

for in 반복문

기본의 for 반복문을 간편하게 압축한 것 

          -- 예제 --

          var array = ['포도', '사과', '바나나', '망고'];

		      for (var i in array) {
			      alert(array[i]);
		      }

for of 반복문

ECMAScript 6 부터 추가된 것으로, 기존의 for in 반복문이 배열의 인덱스를 사용했다면 for of 는 배열의 요소를 직접적으로 사용 가능하다

배열

        var [배열명] = [1, "안", "녕하세요", true]
이런 식으로 선언하며 배열 내부에 들어가는 각 자료형은 변수와 마찬가지로 동적으로 결정되니 더욱 간편하다. '[배열명].length' 속성으로 배열의 사이즈를 구해 반복문과 결합하여 사용할 수도 있고, '[배열명].push([추가할 요소])' 메서드로 배열에 원하는 요소를 추가할 수도 있다. 이것은 특히 JAVA 에서 배열의 크기 변경은 불가능했던 것과 비교하면 매우 편리하다고 할 수 있다
