pre 태그

본래 html 문법은 모든 띄어쓰기나 개행을 '띄어쓰기 한 칸' 으로 인식한다. 그래서 p, br 태그를 사용하는데, 경우에 따라 특정 단락이나 문서내용을 그 상태 그대로 통째로 출력하고 싶을 때는 'pre' 태그를 사용한다
pre 태그로 감싼 내용은 여러 개의 띄어쓰기나 \n 으로 개행된 것 모두 그대로 출력한다

code 태그

pre 태그와 마찬가지로 문단 내용을 있는 그대로 출력한다. 차이는 의미론적인 측면에 존재하는데, pre 태그가 텍스트를 통째로 출력하는 한편, code 태그는 컴퓨터 프로그래밍 코드를 그대로 출력한다
