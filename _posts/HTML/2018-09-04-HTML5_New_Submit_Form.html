HTML5 - form_input

<form 태그(제출)의 input 태그(사용자입력폼)의 새로 추가된 속성 및 type 속성(사용자입력폼의 형식)의 새로운 값>

  • autocomplete 속성: 브라우저 자동완성기능. on 또는 off 할수 있으며 전체 또는 부분적으로 지정할 수 있음
  • placeholder: 사용자입력을 위한 힌트가 나타남
  • autofocus: 최초 해당 페이지 로드 시 자동으로 포커스가 맞춰짐
  • type 속성의 number 값: 텍스트박스에 오직 숫자만 입력가능
  • min, max 속성: 입력되는 최소값과 최대값을 제한

    • 유효하지 않은 수를 입력하고 제출하면, 그것을 거부하고 에러메시지가 출력된다

기타 type 속성값: color, date(날짜), datetime(국제표준시), datetime-local(지역시), email(이메일정보입력), month, number, range(숫자범위_슬라이드형식), search, tel, time, url, week ..

HTML5 에서 추가된 속성 및 기능들은 이용자의 편의성에 중점을 두고 만들어졌다. 특히 모바일에서의 편의성이 많이 향상되었다(숫자입력, 이메일입력, 전화번호입력, url입력 등을 위한 키보드)
