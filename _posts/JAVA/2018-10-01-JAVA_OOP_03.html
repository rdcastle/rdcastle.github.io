  1. 상속

    상속은 부모클래스의 전체 내용을 자식 클래스가 물려받는 일종의 '확장' 개념이다. 결과적으로 적은 양의 코드로 새로운 클래스를 작성할 수 있으며 중복이 제거되고 코드 재사용성이 높아진다.

    상속은 'extends' 키워드로 명시한다. 형식은 다음과 같다 

                class [자식클래스명] extends [부모클래스명] {
              // 코드 내용
            }
