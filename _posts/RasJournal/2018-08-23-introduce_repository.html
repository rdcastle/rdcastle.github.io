개요

저장소는 두 개를 운영한다.
하나는 공부한 내용들을 업로드하는 기록용 블로그 형식이고(RDcastle의 기술공부블로그),
다른 하나는 공부한 것들을 기반으로 실습하는 형식이다.(RDcastle의 실습공간)

RDcastle의 기술공부블로그

현재 상황

다른 뛰어나신 분들이 만든 지킬 테마 중 가장 심플해 보이는걸로 Fork 해와서 일단 포스트하는 중이다.
포스팅 형식은 기본 마크다운으로 하라고 되어있는데, 일단 HTML 에 숙달되고 싶기도 하고 아직 뭣도 모르는 상태에서 (편리하답시고) 마크다운 문법부터 사용하다 보면 HTML 공부하는데 헷갈리기만 할 것 같아서 (느리지만)철저히 HTML 문법에 맞춰 작성할 것이다. 지금 이 글도 HTML 형식에 맞춰서 작성중인데, 아직 아는게 없어서 굉장히 중구난방이다. 의미론적인 정보?를 일목요연하게 알아볼 수 있도록 작성해야 한다는데, 지금은 그럴 능력이 없다...
글의 기본 레이아웃이나 구조를 이 테마에 맞게 작성하려면 마크다운 형식의 문법으로 해야 할것 같은데 확장자를 ~.md 가 아니라 ~.html 로 만들었는데도 테마 형식대로 적용되기는 한다(왜그런지는 모르겠다).

공부 순서

학원 커리큘럼에 따라서 c, c++, java, python, 운영체제, 네트워크, 알고리즘 등등등.. 지금보면 이게 다 뭔가 싶은 광범위한 분야를 깔짝깔짝 맛보기 식으로 공부하긴 했는데(비전공자에 생짜무식이라 학원 팔이에 그대로 따라갔다..), 완벽하진 않아도 최소한 무엇이 어디에 있는지 정도는 알게 되었다. 돈은 아까운 부분이 있지만 어차피 컴퓨터 문외한의 기회비용이라 생각하고, 또 있으면 도움이 되는 지식들이라서 이 블로그에 하나하나 기록해 놓고 나중에 필요할때 찾아서 더 공부하려고 한다.

한편으로 백엔드 웹개발자를 목표로 삼고 관련지식 공부를 집중적으로 하고자 하는 만큼, 프론트 단의 html, css, javascript, 백 단의 java, jsp, DB(오라클, Mysql) 등의 기술을 순차적으로 공부해 나갈 것이다. 먼저 egoing (느님)의 아주 친절한 커리큘럼 과정을 따라가며, 이후에 세부적인 것들은 스스로 찾아서(구글링, 책) 공부할 것이다. 라이브러리나 프레임워크, 그 외 세부적인 사항들은 공부하면서 필요에 의해 방향성을 수정해가며 공부한다. 또 10월부터 웹개발 국비지원 과정을 들어가기 때문에 병행하면서 블로그 포스팅을 할것 같다. 일정 시점부터는 실습공간에 관련 프로젝트도 올릴 것 같다.

블로그 목표!

최종적으로는 내 입맛에 맞는 형식과 디자인의 블로그를 만들 것이다(댓글이나 알림, 구글검색 최적화 등등의 부가기능들도 적용하면 좋겠다). 지금은 다른 사람의 테마를 빌려와서 허술하게 시작하기는 하지만, 공부하면서 하나하나 수정할 것이다. 그 과정 중에서 과거에 작성했던 글도 다시금 예쁘게 수정하는 일이 있을 것 같다. 각각의 공부내용들은 주제별로 카테고리화해서 알아보기 쉽게 구성할 것이다.

RDcastle의 실습공간

일단은 생코에서 배운것들 위주로 만들고, 어느 시점부터는 그것들을 기반으로 내 입맛에 맞는 별도의 프로젝트도 진행할 것이다. 나중에 국비교육 과정에서의 프로젝트도 별도로 이곳에 만들수도 있을 것 같다 (오픈소스에 참여할만한 여건이 되면 그것도 같이 해볼지도?).
