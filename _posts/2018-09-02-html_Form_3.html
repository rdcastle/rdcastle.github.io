form tag -> input -> type="file"

input 태그의 type 속성 중의 하나이다. 정해진 경로의 서버로 파일을 선택하여 전송하기 위한 기능. 선택하고 나면 파일명이 옆에 나타난다.

예제


예제코드

        
          <form action="https://localhost/index.php" method="post" enctype="multipart/form-data">
            <label for="RDcastle_img">RDcastle 이미지 :</label> <input type="file" name="RDcastle_img" id="RDcastle_img"><br>
            <input type="submit">
          </form>

* form 태그 속성으로 포함된 enctype="multipart/form-data" 부분은 일단 외워둘 것
