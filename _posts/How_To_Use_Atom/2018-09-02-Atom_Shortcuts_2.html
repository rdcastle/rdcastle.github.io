문자열 검색

  1. Ctrl + d

    2. 해당 문자열에 커서를 올려놓고 Shortcuts를 반복해서 누르면 같은 문자열이 하나씩 추가로 선택된다. 원하는 만큼 선택하고 수정 가능.

  2. Ctrl + f

    3. 현재 페이지 내에서 입력받은 문자열을 모두 찾아준다. 수정할 문자열을 입력하면 한 번에 바꿔준다.

  3. Ctrl + Shift + f

    4. 폴더를 선택하고 단축키를 눌러 검색하고자 하는 문자열을 입력하면, 폴더 내 모든 파일 중 일치하는 것을 나타낸다. 원하는 문자열이 포함된 파일을 찾을 때 사용
