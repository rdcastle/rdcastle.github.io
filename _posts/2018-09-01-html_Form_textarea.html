Textarea

input 태그가 한 줄의 text 입력 칸을 만든다면, textarea 태그는 여러 줄을 입력하는 칸을 만든다(별도의 태그임)
여러 속성들이 사용 가능하며, cols 속성은 열, rows 속성은 행을 의미한다. 태그 내용을 입력해두면 default value 가 된다

사용 예


    <textarea name="textarea" id="" cols="50" rows="50">default value</textarea>
