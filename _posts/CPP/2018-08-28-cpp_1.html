CPP 와 C 의 차이점

  1. 반환형 생략가능 여부

    2. 그럴 일은 잘 없지만, C 언어에서는 함수의 반환형이 생략 가능하며 생략 시에는 디폴트 int 형으로 반환된다(이것을 고려해야 한다)
    반면, C++ 언어에서는 함수의 반환형이 생략될 수 없다. 무조건 반환형을 선언해 줘야 한다!!

  2. 지역변수 선언 후 다른 구문 수행 후 또다른 지역변수 선언하기

    3. 본래 변수 선언 및 초기화는 함수의 최상부에 가장 먼저 이행해줘야 하지만, C 언어에서는 빌드가 되기는 한다 이는 비주얼 스튜디오에서만 편의성을 고려해 가능한 것 뿐이고 기본적인 C 문법에서는 불가능하다
    그러나 C++ 언어는 어느 위치에서나 변수 선언 및 초기화를 해도 문법적으로 상관 없다

  3. 기타 여러가지

    4. 레퍼런스나 클래스, 동적할당 방식 등 C 언어에서 개선되거나 추가된 기능들이 존재한다

CPP 개요

특히 객체지향 개념의 추가를 주목해야 하며 이것에 대해 인식하는 것이 중요하다
(C 언어는 절차지향 - 순서를 중시 - 언어이다)

객체 - Object

사물화, object화 된 모든 것, 모든 개념
특정 객체의 일련의 논리적 속성 및 메소드를 구성하는데 있어서 C 언어는 절차적으로 구현해야 하기 때문에 복잡해지는 경향이 있으나 C++ 객체지향 언어는 이러한 객체 전반을 하나의 object 로 묶어서 보다 일관성있고 편리하며 직관적으로 표현할 수 있다
